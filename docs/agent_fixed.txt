### **Part 3: Fixes for Your Current Codebase**

Based on the files you shared (`agents_information.txt`), your current codebase needs these specific fixes to be "Production Ready."

#### **1. Fix the "Theme Switcher" Flicker**

  * **Problem:** The current snippet might cause a "white flash" on dark mode reload.
  * **Fix:** Move the theme detection script **into the `<head>` tag** of your `index.html` (before React loads).
    ```javascript
    // Add this inside <head> in index.html
    <script>
      if (localStorage.theme === 'dark' || (!('theme' in localStorage) && window.matchMedia('(prefers-color-scheme: dark)').matches)) {
        document.documentElement.classList.add('dark')
      } else {
        document.documentElement.classList.remove('dark')
      }
    </script>
    ```

#### **2. Enforce "Rate Limiting" on Public Agents**

  * **Problem:** If you make this free, bots will drain your API credits.
  * **Fix:** Add a simple middleware in your backend (FastAPI/Express).
      * **Logged Out Users:** 3 runs per day (track via IP address).
      * **Logged In Users:** 50 runs per day.
      * **Return:** `429 Too Many Requests` status code.

#### **3. Standardize the "Agent Response" JSON**

  * **Problem:** If Agent A returns `{ text: "..." }` and Agent B returns `{ output: "..." }`, your frontend code gets messy.
  * **Fix:** Enforce a strict schema for **all** agents:
    ```json
    {
      "status": "success",
      "data": {
        "markdown": "## Result...",
        "files": [ {"name": "report.pdf", "url": "..."} ],
        "meta": { "cost": 0.002, "time_ms": 450 }
      }
    }
    ```

#### **4. Add "Sandbox Mode" for Demos**

  * **Problem:** Users won't sign up if they can't test it.
  * **Fix:** For the **top 3 agents** (like Resume Optimizer), allow 1 free run *without* login.
      * **UI:** Show a blurred result or the first 50% of the output, then show a "Login to see full report" button.

#### **5. UI Polish: "Terminal" vs "Card" View**

  * **Problem:** The blueprints mention both "Terminal style" and "Card style."
  * **Fix:**
      * **Dashboard:** Use **Card Grid** (clean, easier to browse 20 agents).
      * **Agent Run Page:** Use **Split Screen**. Left side = Inputs. Right side = Terminal-style output window (monospace font, dark background) to look like a "DevOps Console."
